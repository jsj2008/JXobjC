Dictionary

Superclass:

Introduction

Stores associations of keys to values. For each key, a value is associated, and with that key the value may be accessed.

atKey:

- atKey:aKey;
Parameters
aKey

Key to find value for.

Discussion

Retrieves the value at the specified key.

The Key is located in the dictionary using isEqual:. If the key is not found, returns nil.

atKey:ifAbsent:

- atKey:aKey ifAbsent:exceptionBlock;
Parameters
aKey

Key to find value for.

exceptionBlock

Block to be evaluated and value returned if key is not found.

Discussion

Retrieves the value at the specified key, evaluating a specified block if no such key is found in the dictionary and returning its result.

atKey:put:

- atKey:aKey put:anObject;
Parameters
aKey

Key to insert value for.

anObject

Value to associate with aKey.

Discussion

Inserts the specified value for the specified key.

If a key that isEqual: to the specified key is found in the dictionary, returns the previous value for that key. Otherwise, returns nil.

copy

- copy;
Discussion

Copies the dictionary. The associations are copied but the keys and values point to the same objects as in the original.

deepCopy

- deepCopy;
Discussion

Copies the dictionary, each key and value being sent a deepCopy message in turn.

eachKey

- eachKey;
Discussion

Returns a sequence of each key in the dictionary.

eachValue

- eachValue;
Discussion

Returns a sequence of each value in the dictionary.

emptyYourself

- emptyYourself;
Discussion

Empties the dictionary of all associations.

hash

- (uintptr_t)hash;
Discussion

Returns the unique hash of the dictionary, combining the hash of each key and value.

includesKey:

- (BOOL)includesKey:aKey;
Parameters
aKey

Which object's presence as a key in the dictionary should be checked.

Return Value

YES if aKey is in the dictionary, NO if not.

Discussion

Queries the dictionary to find out whether it includes an object as a key.

isEmpty

- (BOOL)isEmpty;
Return Value

YES if it is empty, NO if it is not.

Discussion

Queries the Dictionary on whether it is empty.

isEqual:

- (BOOL)isEqual:aDic;
Parameters
aDic

Dictionary to compare with.

Discussion

Compares the dictionary to another.

If each key and value in turn responds with YES to isEqual:, returnsYES.

keysDo:

- keysDo:aBlock;
Parameters
aBlock

Block to be called with each key as an argument.

Discussion

Calls a block with each key in the dictionary, passing a single parameter to the block, the current key.

removeKey:

- removeKey:key;
Parameters
key

Key whose according association is to be removed.

Discussion

Removes the association keyed to the specified key and returns its value.

If no such key is found, the NotFound exception is signaled.

removeKey:ifAbsent:

- removeKey:key ifAbsent:aBlock;
Parameters
key

Key whose according association is to be removed.

aBlock

Block to be evaluated & value returned if key is not found.

Discussion

Removes the association keyed to the specified key and returns its value. If such a key is not found, returns the result of the specified block.

size

- (unsigned)size;
Return Value

The number of key-value associations.

Discussion

Query the size of the dictionary in terms of the count of key-value associations.
