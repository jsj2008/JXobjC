OrdCltn

Superclass:
Cltn
Declared In:
OrdCltn.h

Introduction

An Ordered collection of objects.

Discussion

Stores objects in an ordered form similar to the C++ STL's Vector. Objects at any specified index may be retrieved, removed, or inserted. The index begins at 0 for the first object, and ends at size - 1 for the last.

copy

- copy;
Discussion

Copies the OrdCltn. The objects themselves are not copied, only their identifiers, so the contents are 100% identical to the original OrdCltn.

deepCopy

- deepCopy;
Discussion

Copies the OrdCltn, sending a deepCopy message to each object contained.

do:

- do:aBlock;
Parameters
aBlock

Block to be called with each object as its argument.

Discussion

Calls a block with each object in the OrdCltn, passing in turn each object as the single parameter to the block.

do:until:

- (id) do:aBlock until:(BOOL *)flag;
Parameters
aBlock

Block to be called with each object as its argument.

flag

Pointer to a boolean value, which halts further iteration when it becomes YES.

Discussion

Calls a block with each object in the OrdCltn, until a BOOL becomes YES.

isEmpty

- (BOOL)isEmpty;
Return Value

YES if it is empty, NO if it is not.

Discussion

Queries the OrdCltn on whether it is empty.

lastOffset

- (unsigned)lastOffset;
Return Value

Offset of the last element if OrdCltn is not empty, -1 if it is.

Discussion

Requests the offset of the last element.

reverseDo:

- reverseDo:aBlock;
Parameters
aBlock

Block to be called with each object as its argument.

Discussion

Calls a block with each object in the OrdCltn, starting with the last and proceeding backwards to the first.

size

- (unsigned)size;
Discussion

Queries the OrdCltn, returning the number of objects stored in it.
