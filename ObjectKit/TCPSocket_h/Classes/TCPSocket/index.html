bindToHostname:port:

Binds to the specified hostname and port. 

- (unsigned short)bindToHostname:(String *)host port:(unsigned short)port;
Parameters
host

String hostname to bind to.

port

Unsigned short port to bind to.

Discussion

A port of 0 will result in the choosing of a random port. That port number will be returned. If binding fails, signals an exception detailing the problem.

connectToHostname:port:

Connects to the specified hostname and port. 

- (id)connectToHostname:(String *)host port:(unsigned short)port;
Parameters
host

String hostname to connect to.

port

Unsigned short port to connect to.

Discussion

If connection fails, an exception will be signaled detailing the problem. Otherwise, you may read or write to the socket or place it into a RunLoop.

listen

Begins listening for connections. 

- listen;
Discussion

After this, the socket can be read from or placed into a RunLoop. If listening fails, signals an exception.
