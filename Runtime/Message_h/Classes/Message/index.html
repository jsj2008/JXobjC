Message

Superclass:
Object
Declared In:
Message.h

Introduction

Message represents a message made static; it is a reification of the activity of sending a message to an object. This is used mainly for forwarding of messages by doesNotUnderstand:.

selector:dispatch:args:

+ selector:(SEL)sel dispatch:(ARGIMP)disp args:(void *)arg;
Parameters
sel

Selector name.

disp

Message dispatcher. This is a compiler-generated function which will decode the arguments and execute the associated IMP.

arg

Argument structure. This is a struct beginning with the return type, followed by each argument in sequence.

Return Value

A new message object initialised with the specified parameters.

Discussion

Creates a new message object.
